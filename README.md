# Cube Crafter Website

This is the official website for **Cube Crafter**, a comprehensive Discord bot designed for Rubik's cube enthusiasts. 

The website showcases the bot's features, including official WCA scramble generation, image visualization, and advanced timing capabilities.

## Related Repositories

- **Discord Bot Source Code:** [rubik-bot](https://github.com/timmy0402/rubik-bot)

## Features

- **Official Scrambles:** Supports all WCA competition puzzles with official algorithms.
- **Image Generation:** Visualize every face of your scramble directly in Discord.
- **Advanced Timer:** Measure and track your progress with an integrated database.
- **Community Focused:** Easy integration into any Discord server.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org) (App Router)
- **Styling:** CSS Modules
- **Icons:** Custom SVG Components
- **Optimization:** Next.js Image optimization with static imports and blur placeholders.
